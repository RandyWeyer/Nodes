## Specifications

  * Create Cards
    * Draggable
    * Has a title and description
    * Optional url for image
    * Change color
    * Removable
  * Lines can connect Cards
  * Save Cards and Lines
  * Resize canvas based on window size
  * toolbar
    * Home
    * Save
    * Load
    * Contact
    * About
      * Go to GitHub Repository
